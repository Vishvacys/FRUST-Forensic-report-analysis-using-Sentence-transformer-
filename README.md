# FRAUST - Forensic Analysis System

AI-powered UFDR forensic data analysis tool built with Flask + React.

## Features
- UFDR file parsing and analysis
- NLP-based chat and call log investigation
- Crypto address and suspicious keyword detection
- Risk scoring and forensic report generation

## Tech Stack
- Backend: Python, Flask, SQLAlchemy, Sentence Transformers, FAISS, SpaCy
- Frontend: React, Vite, Tailwind CSS, Recharts

## Setup

### Backend
    pip install -r requirements.txt
    python app.py

### Frontend
    cd frontend
    npm install
    npm run dev
